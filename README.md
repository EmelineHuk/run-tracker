# Run Tracker

O Run Tracker é uma aplicação web para registrar e acompanhar treinos de corrida.

## Funcionalidades

- cadastro de treinos com data, distância e tempo
- cálculo automático de pace
- histórico de treinos cadastrados
- validação básica de distância

## Tecnologias utilizadas

- React
- Vite
- Vitest
- GitHub Actions

## Como executar o projeto

```bash
npm install
npm run dev
